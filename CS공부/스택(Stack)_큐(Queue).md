# 스택(Stack)
- 입력과 출력이 한 곳(방향)으로 제한
- LIFO (Last In First Out, 후입선출) : 가장 나중에 들어온 것이 가장 먼저 나옴

### 언제 사용?
- 함수의 콜스택, 문자열 역순 출력, 연산자 후위표기법

- 데이터 넣음 : push()
- 데이터 최상위 값 뺌 : pop()
- 비어있는 지 확인 : isEmpty()
- 꽉차있는지 확인 : isFull()
- +SP

- push와 pop 할 때는 해당 위치를 알고 있어야 하므로 기억하고 있는 '스택 포인터(SP)'가 필요함
- 스택 포인터는 다음 값이 들어갈 위치를 가리키고 있음 (처음 기본값은 -1)
